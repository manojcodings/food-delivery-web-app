# 🍔 Food Delivery Web App

A full-stack food delivery web application built using PHP, MySQL, HTML, CSS, and Bootstrap.

## 🚀 Features
- User-friendly UI
- Food ordering system
- Admin dashboard
- Order management
- Responsive design

## 🛠 Tech Stack
- Frontend: HTML, CSS, Bootstrap
- Backend: PHP
- Database: MySQL
